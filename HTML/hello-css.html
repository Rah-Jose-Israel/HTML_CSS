Hello, CSS

CSS lets us style HTML elements. There’s also another page associated with this example.

A CSS “rule” always start with a “selector” that defines which HTML elements it applies to. In this case, we’re trying to style the element. After the selector, we have the “declarations block” inside of some curly braces.
Any “properties” we set in here will affect the element.

List Styles

“cascading” part of CSS

CSS hierarchy for every web page looks like this:

  1. The browser’s default stylesheet
  2. lower-roman
  3. User-defined stylesheets
  4. lExternal stylesheets (that’s us)
  5. Page-specific styles (that’s also us)
  6. Inline styles (that could be us, but it never should be)

    7. page-specific styles
